# bertfamily
用Keras实现更轻量级的bert及其变种


## 使用
安装方式：
```shell
pip install git+https://github.com/xuyingjie521/bertfamily.git
```
理论上兼容Python2和Python3，实验环境是Python 2.7、Tesorflow 1.13+以及Keras 2.2.4（也可在2.2.4、2.3.0、2.3.1、tf.keras下使用，建议试用稳定的2.2.4版本）
